# Pedra, Papel e Tesoura em Python

Jogo clássico de Pedra, Papel e Tesoura feito em Python para rodar no terminal.  
O jogador enfrenta a máquina, com validação de entradas, sorteio aleatório e placar em tempo real.  
A partida termina quando alguém atinge 3 pontos.

## Como executar

1. Certifique-se de ter o Python 3 instalado
2. Clone o repositório ou baixe os arquivos, certifique-se de que o arquivo.json esteja na mesma pasta do arquivo.py
3. No terminal, entre na pasta do projeto:
```
cd jogo.py
```   
Execute o jogo:
```
python jogo.py
